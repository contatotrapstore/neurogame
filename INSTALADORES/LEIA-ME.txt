📋 REQUISITOS DO SISTEMA:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

• Sistema Operacional: Windows 7 ou superior (64-bit)
• Memória RAM: Mínimo 2GB (recomendado 4GB)
• Espaço em Disco: 500MB para o launcher + espaço para jogos
• Internet: Conexão estável para download de jogos


🎮 APÓS A INSTALAÇÃO:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. O BioSync Launcher ficará disponível em:
   • Atalho na Área de Trabalho
   • Menu Iniciar → BioSync Launcher
   • C:\Users\[SeuUsuário]\AppData\Local\Programs\biosync-launcher

2. Na primeira execução:
   • Faça login com seu e-mail/username e senha
   • Navegue pela biblioteca de jogos
   • Clique em um jogo para baixar e instalar

3. Os jogos serão instalados em:
   • Pasta padrão: C:\Users\[SeuUsuário]\AppData\Roaming\biosync-launcher\Jogos\
   • Gerenciado automaticamente pelo launcher

4. Logs para diagnóstico são salvos em:
   • C:\Users\[SeuUsuário]\AppData\Roaming\biosync-launcher\logs\


🔄 ATUALIZAÇÕES AUTOMÁTICAS:

🐛 SOLUÇÃO DE PROBLEMAS:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Se o launcher mostrar tela branca ou não iniciar:

1. Verifique sua conexão com a internet
2. Reinicie o launcher completamente
3. Verifique se o firewall/antivírus está bloqueando
4. Veja os logs em: C:\Users\[Usuário]\AppData\Roaming\biosync-launcher\logs
5. Reinstale se o problema persistir
6. Entre em contato com o suporte
